## 快速使用

### 1. 开发模式
```bash
# 安装 Python 依赖
pip install -r requirements.txt

# 启动后端
python backend/main.py

# 另一个终端：启动 Electron
cd electron
npm install
npm run dev
```

### 2. Windows 打包
```bash
build.bat
```

### 3. GitHub Actions 自动构建
推送标签即可触发：
```bash
git tag v1.0.0
git push origin v1.0.0
```
